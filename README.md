## ✨ Features

- **🌑 Modern Deep Dark Interface:** A beautiful, intuitive, ChatGPT-inspired aesthetic built with PySide6.
- **📊 Live Dashboard:** Real-time tracking of Memory Usage, Processor Load, System Uptime, and Active Processes.
- **🚀 One-Click Boost:** Instantly clears working set RAM from active processes to free up memory without crashing apps.
- **💀 Kill Switch:** Automatically scans and force-terminates any applications stuck in a "Not Responding" state.
- **🛠️ 100+ Advanced Tweaks:** Safely execute deep-system optimizations spanning:
  - Network & Internet 
  - System Cleanup & Cache
  - Windows Explorer & Workflow
  - Visual Effects & Animations
  - Privacy & Telemetry Disablements
  - Windows Services Management
  - Gaming & Xbox Optimizations
  - Power & Performance Plans
- **⚙️ Seamless Windows Integration:**
  - **Context Menu:** Add the optimizer directly to your desktop Right-Click menu.
  - **Startup:** Set the app to run silently in the system tray on Windows boot.
  - **Auto-Clean:** Automatically clears RAM in the background every 10 minutes.
